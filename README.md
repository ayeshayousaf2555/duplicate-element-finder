# 🔍 Duplicate Element Finder

A beginner-friendly **C++ program** that detects and prints duplicate elements in an array.  
This project demonstrates basic array handling, nested loops, and comparison logic.

---

## 📌 Features
- Accepts user input for number of elements.
- Reads array elements from the user.
- Compares each element with the rest to find duplicates.
- Prints duplicate values once.

---

## ⚙️ Implementation Details
- Uses a fixed-size array (`size = 100`).
- Input: number of elements (`n`) and the elements themselves.
- Logic: nested loop compares each element with subsequent elements.
- Output: duplicate elements displayed on the console.
